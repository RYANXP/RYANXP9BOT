
## Clone Este github

```bash
> git clone https://github.com/RYANXP/RYANXPBOT
```

## Instale com esses  comandos:
Manda !help ou !menu apos  escanear o qr

```bash
> cd RYANXP
> bash install.sh
```

### Use Para  escanear qr
```bash
> npm start
```
